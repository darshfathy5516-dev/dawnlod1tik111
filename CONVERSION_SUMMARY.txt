## Deployment Instructions

### For Standard Shared Hosting (iPage, etc.):

1. **Upload the `dist` folder contents:**
   ```
   dist/
   ├── index.html
   ├── assets/ (CSS & JS files)
   ├── en/
   ├── ar/
   ├── es/
   ├── fr/
   ├── de/
   └── pt/
   ```

2. **Upload to your web host's public_html or www folder**

3. **That's it!** No Node.js, no build step needed on the server.

### Build Command (if needed):
```bash
npm install
npm run build
```

The built files will be in the `dist/` folder.

### Development Server:
```bash
npm install
npm run dev
```
Visit http://localhost:3000
